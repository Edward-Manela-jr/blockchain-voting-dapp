MetaMask Setup Guide
Overview

This project uses MetaMask to interact with the blockchain smart contracts deployed locally using Hardhat.

MetaMask allows users to:

Sign blockchain transactions

Connect the frontend to the smart contract

Test interactions with the blockchain

1. Install MetaMask

Download the extension from the official website:

https://metamask.io

Supported browsers include:

Chrome

Firefox

Edge

Brave

Steps:

Install the MetaMask extension.

Click Create a Wallet.

Set a password.

Save the Secret Recovery Phrase securely.

⚠️ Never share the Secret Recovery Phrase with anyone.

2. Add the Local Hardhat Network

To interact with the local blockchain, MetaMask must connect to the Hardhat local network.

Open MetaMask and navigate to:

Settings → Networks → Add Network

Enter the following details:

Network Name: Hardhat Localhost
RPC URL: http://127.0.0.1:8545
Chain ID: 31337
Currency Symbol: ETH

Save the network.

3. Start the Local Blockchain

Before interacting with MetaMask, start the local blockchain using Hardhat.

Run:

npx hardhat node

This starts a local Ethereum network and generates test accounts.

Example output:

Started HTTP and WebSocket JSON-RPC server at http://127.0.0.1:8545/

Accounts
========
Account #0: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266
Private Key: 0xac0974bec39a17e36ba4a6b4d238ff...
4. Import a Hardhat Test Account into MetaMask

To interact with the blockchain, import one of the Hardhat accounts.

Steps:

Open MetaMask

Click the Account icon

Select Import Account

Paste the Private Key from the Hardhat terminal

Example:

Private Key: 0xac0974bec39a17e36ba4a6b4d238ff...

Once imported, MetaMask will show a balance of:

10000 ETH

These are test tokens for development purposes.

5. Confirm Network Connection

Ensure MetaMask displays:

Network: Hardhat Localhost
Balance: 10000 ETH

This confirms the wallet is connected to the local blockchain.

Challenges Encountered During Setup
Issue 1: MetaMask Not Connecting to Local Network

MetaMask may fail to connect if the Hardhat node is not running.

Solution

Ensure the local blockchain is running:

npx hardhat node

MetaMask will then connect successfully.

Issue 2: Transactions Not Appearing

If transactions do not appear in MetaMask, it is often because the wrong network is selected.

Solution

Ensure MetaMask is set to:

Hardhat Localhost

instead of the default Ethereum network.

Issue 3: Account Shows Zero Balance

If the imported account shows no balance, it may not be one of the test accounts generated by Hardhat.

Solution

Import one of the private keys shown in the Hardhat terminal after running:

npx hardhat node
